INSTRUCCIONES:

1) Desde la carpeta src se debe ejecutar el Makefile con el comando make en la terminal.
2) Luego de esto, se crearán los ejecutables en la carpeta bin. Por lo tanto, para correrlos se debe ejecutar considerando ese directorio.
3) Para ejecutar cada main de la estructura de dato respectiva, se debe considerar los siguientes nombres del ejecutable: 
    3.1) BST (Binary Search Tree)
    3.2) CHT (Closed Hashing Table)
    3.3) OHT (Open Hashing Table)
    3.4) SET (Set STL)
    3.5) USET (Unordered set STL)
4) Luego, se debe llamar al archivo desde el directorio "data" junto a cada ejecutable. Este es "universities_followers_2022_v1".
5) Por último, los resultados se encontrarán en results en formato .txt y graficados en el Jupyter Notebook Plots.ipynb
